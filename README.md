# ASCII Lab

A simple web app for ASCII conversions and an ASCII table.

## Features
- Text to ASCII and ASCII to text
- Binary, hex, and octal conversions
- Clean, responsive UI
- Dark mode and language toggle

## Pages
- `index.html` - Landing page
- `convert.html` - Converter tools
- `table.html` - ASCII table

## Getting Started
1. Open `index.html` in your browser.
2. Use the navigation to access converters or the ASCII table.

## Project Structure
- `app.js` - UI logic and converters
- `styles.css` - Styling
